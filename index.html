🦙 Setup Ollama (Mode Lokal)

Install dan jalankan Ollama di komputer kamu:

# Install
curl -fsSL https://ollama.ai/install.sh | sh
# Windows: https://ollama.ai

# Jalankan dengan CORS enabled (WAJIB)
OLLAMA_ORIGINS="*" ollama serve

# Pull model
ollama pull llama3.2        # 2GB
ollama pull qwen2.5:3b      # 1.9GB - ringan
⚠️ PENTING: Jalankan dengan OLLAMA_ORIGINS="*" ollama serve
