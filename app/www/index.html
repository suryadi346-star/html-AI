🦙 Setup Ollama (Mode Lokal)

Install dan jalankan Ollama di komputer kamu terlebih dahulu:

# 1. Install Ollama
curl -fsSL https://ollama.ai/install.sh | sh
# Windows: download dari https://ollama.ai

# 2. Jalankan dengan CORS enabled (WAJIB untuk browser)
OLLAMA_ORIGINS="*" ollama serve

# 3. Pull model yang diinginkan
ollama pull llama3.2        # 2GB - umum
ollama pull qwen2.5:3b      # 1.9GB - ringan & cepat
ollama pull mistral         # 4.1GB - powerful
ollama pull phi3.5          # 2.2GB - Microsoft
⚠️ PENTING: Browser membutuhkan CORS header. Jalankan Ollama dengan OLLAMA_ORIGINS="*" ollama serve. Di Windows: set variabel env OLLAMA_ORIGINS=* sebelum menjalankan.

Setelah Ollama berjalan, klik Coba Hubungkan untuk mendeteksi model secara otomatis.
